### Description

EVal evaluates math expressions like 2 + 3 / ( 99.5 - 10 ). Whole code is in a .bas file and there is a test application (see Screenshot). Operations availables are +, -, /, * and to make powers (^) usage character e (Ex: 2e4 = 16).
